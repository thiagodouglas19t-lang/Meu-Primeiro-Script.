# Academia Thiago — Central de Estudos

Ambiente de estudo virtual criado para estudar programação, inteligência artificial, edição de vídeo, empreendedorismo, comunicação, raciocínio lógico e organização pessoal.

## O que tem

- Dashboard com XP, nível e progresso
- Missões diárias
- Áreas de estudo
- Projetos reais
- Biblioteca de notas e prompts
- Timer de foco
- Salvamento local pelo navegador

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
